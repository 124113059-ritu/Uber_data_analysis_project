# 🚖 Uber Data Analysis Project

A comprehensive Exploratory Data Analysis (EDA) project on Uber trip data using Python. The objective of this project is to uncover meaningful insights about trip patterns, ride demand, time-based trends, and user behavior through data visualization and statistical analysis.

---

## 📌 Project Overview

This project focuses on analyzing Uber trip data to answer important business questions such as:

- When are Uber rides most frequent?
- Which days have the highest demand?
- How do trips vary across different months and hours?
- What trends can be observed from the dataset?

The analysis is performed using Python and popular data science libraries.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Time-based Analysis
- Monthly, Weekly, and Daily Ride Trends
- Interactive Data Visualization
- Statistical Insights

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/124113059-ritu/Uber_data_analysis_project.git
```

Navigate to the project directory:

```bash
cd Uber_data_analysis_project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
